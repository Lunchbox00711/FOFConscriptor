Import Ratings Data

To import ratings data, you need to export the scout data from Front Office Football using the option at the bottom of the Almanac menu in FOF7 version 7.0b+. This will be in a .csv file ("draft_personal.csv" most likely). When you import this file, the player bars as well as the data from your interviews will be visible in all your player lists.
